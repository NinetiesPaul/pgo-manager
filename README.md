##PGO-Manager


Simple web system for Pokemon PVP Team management

Project running on:
* ```PHP 7.2```
* ```MySQL 5.2```

Composer packages used:
* ```Phinx```
* ```PHP-Cs-Fixer```
* ```Simple-PHP-Router```
* ```VLUCAS's PHPDOTENV```

Before running the project:
* Install PHP 7.0 or higher
* Install Composer
